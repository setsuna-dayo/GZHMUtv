# GZHMUtv - 校内网免流量电视的第三方PC客户端

## 简介

由于学校校园网的网络电视年久失修，依然在使用Flash，于是随手写了个小工具。

支持校内网**免流量**电视直播和电影库点播。

！arm32/64的Windows设备未测试！

设计了遥控器和电视机，模拟电视机的感觉（莫名的怀旧感）

播放窗口支持自动拉伸。
